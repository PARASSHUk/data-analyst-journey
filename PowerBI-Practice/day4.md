## New Visualization Types Learned Today

### Line and Clustered Column Chart
- Combines bar chart and line chart in one visual
- Columns show actual values like sales or revenue
- Line shows another metric like profit or target
- Use to compare two related metrics together

### Scatter Chart
- Finds relationship between two numeric variables
- Each dot is one data point
- Identifies positive negative or no correlation

### Donut Chart
- Shows how each category contributes to 100%
- Same as pie chart with hollow center
- Center space shows total value

### Card
- Shows ONE important KPI number
- Most prominent visual on dashboard
- User notices immediately

### Table
- Shows exact row level values
- Use when detailed data is needed
- Supports conditional formatting
