## All Formulas Learned This Week

| Formula | What it Does |
|---------|-------------|
| =SUM(range) | Adds all numbers |
| =AVERAGE(range) | Finds average |
| =COUNT(range) | Counts numbers |
| =VLOOKUP(value,range,col,FALSE) | Looks up value left to right |
| =XLOOKUP(value,range,return) | Looks up value any direction |
| =INDEX(range,MATCH(value,range,0)) | Gets value using position |
| =IF(condition,true,false) | Checks a condition |
| =COUNTIF(range,criteria) | Counts based on condition |
| =SUMIF(range,criteria,sum) | Adds based on condition |
| =LEN(text) | Counts characters |
| =LEFT(text,number) | Extracts from left |
| =RIGHT(text,number) | Extracts from right |
| =TEXT(H2,"dd/mm/yyyy") | Converts date to text |
| =TRIM(text) | Removes extra spaces |
| =LOWER(text) | Converts to lowercase |
| =UPPER(text) | Converts to uppercase |
| =PROPER(text) | Capitalizes first letter |
| =CONCATENATE(A2," ",B2) | Joins two columns |
| =DAYS(end,start) | Calculates total days |
| =NETWORKDAYS(start,end) | Calculates working days |
| =MAX(range) | Finds highest number |
| =MIN(range) | Finds lowest number |
