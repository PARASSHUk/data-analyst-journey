## Formulas and Features I Learned

- =VLOOKUP(value, range, column, FALSE)
  searches only left to right
  needs column number manually

- =XLOOKUP(value, lookup range, return range)
  searches in any direction
  easier and more flexible than VLOOKUP
  no need to count column numbers

- Conditional Formatting
  highlights cells based on rules
  example: highlight all sales above 5000 in green
  example: highlight duplicates in red

- Charts
  Bar Chart    → comparing categories
  Line Chart   → showing trends over time
  Pie Chart    → showing percentage breakdown

## Difference Between VLOOKUP and XLOOKUP

| Feature | VLOOKUP | XLOOKUP |
|---------|---------|---------|
| Direction | Left to right only | Any direction |
| Column number | Needed manually | Not needed |
| If value not found | Shows error | Can show custom message |
| Ease of use | Medium | Easy |
| Version | Old formula | New formula |
