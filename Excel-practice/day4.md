## Formulas I Learned

- =LOWER(text)                        converts text to lowercase
- =UPPER(text)                        converts text to uppercase
- =PROPER(text)                       converts first letter to capital
- =CONCATENATE(A2," ",B2)            joins first name and last name
- =A2&" "&B2                          shorter way to concatenate
- =DAYS(end_date, start_date)         calculates total days between dates
- =NETWORKDAYS(start_date, end_date)  calculates working days only
- =LOOKUP(value, range)               looks up a value in a range
