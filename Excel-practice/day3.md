## Formulas I Learned

- =LEN(text)                        counts number of characters
- =LEFT(text, number)               extracts characters from left
- =RIGHT(text, number)              extracts characters from right
- =TEXT(H2,"dd/mm/yyyy")            converts date to text format
- =TRIM(text)                       removes extra spaces
- =XLOOKUP(value, range, return)    advanced version of VLOOKUP
- =MAX(range)                       finds highest number
- =MIN(range)                       finds lowest number
- =MATCH(value, range, 0)           finds position of a value
- =INDEX(range, MATCH(value,range,0))  gets value using position
